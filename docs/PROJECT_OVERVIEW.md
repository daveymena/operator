# PROYECTO UNIFICADO — SISTEMA SUPREMO DE IA + CONTROL TOTAL + AUTOMATIZACIÓN

```
   ╔══════════════════════════════════════════════════════════════════════╗
   ║         UNIFICACIÓN ABSOLUTA: IA + PC CONTROL + WEB AUTOMATION     ║
   ║              UN SOLO SISTEMA - TODOS LOS PODERES                   ║
   ╚══════════════════════════════════════════════════════════════════════╝
```

Este sistema es la **fusión definitiva** de tres mundos: inteligencia artificial
avanzada (Hermes), control físico absoluto del PC (OpenCode) y automatización
web profesional (Facebook + WhatsApp). Trabajan como un solo organismo.

### OPENCODE — Control Físico Total del PC
```
COMANDOS DIRECTOS (vía PC Agent):
├── screenshot(quality, scale) → Captura JPEG optimizada (50-300ms)
├── mouse_move(x, y) → Precisión absoluta en píxeles (~50ms)
├── mouse_click(button) → left/right (~80ms)
├── mouse_double_click() → Doble click
├── mouse_scroll(clicks) → Scroll vertical
├── drag_and_drop(x1,y1,x2,y2) → Arrastrar y soltar
├── keyboard_type(text) → Escritura instantánea (~5ms/char)
├── keyboard_press(key) → ENTER, TAB, ESC, F1-F12, etc.
├── keyboard_shortcut(modifiers, key) → Ctrl+C, Alt+Tab, etc.
├── get_clipboard() → Lee portapapeles
├── set_clipboard(text) → Escribe portapapeles
├── sysinfo() → RAM, CPU, SO, hostname
├── list_windows() → Todas las ventanas abiertas
├── list_apps() → Apps instaladas
├── powershell(script) → CUALQUIER script PowerShell
├── cmd(command) → CUALQUIER comando CMD
├── read_file(path) → Lee cualquier archivo
├── write_file(path, content) → Escribe cualquier archivo
├── list_dir(path) → Lista directorios
├── open_url(url) → Abre URL en navegador
├── open_file(path) → Abre archivo con app asociada
├── focus_window(pid) → Enfoca ventana específica
├── notify(message, title) → Notificación en pantalla
└── wait(ms) → Pausa controlada
```

## 🚀 CÓMO USAR EL PODER

### Para DOMINAR el PC con IA:
```cmd
cd C:\Users\ADMIN\Music\proyecto-unificado
hermes.bat
```
→ Hermes se inicia con 12 superpoderes para controlar tu PC

### Para AUTOMATIZAR Facebook:
```cmd
node facebook-automation/scripts/ads/crear-borradores-adsmanager.mjs
```
→ Crea campañas en Ads Manager automáticamente

### Para VENDER por WhatsApp:
```cmd
cd whatsapp-bot
npm start
```
→ Bot de ventas con IA conectado a WhatsApp
