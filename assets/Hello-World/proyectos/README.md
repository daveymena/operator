# Mis Proyectos

Esta carpeta es tu espacio para crear proyectos dentro del ecosistema OpenCode en Replit.

## Cómo empezar un proyecto

Puedes pedirle directamente a OpenCode (en el chat) cosas como:

- "Crea un proyecto de React con Vite en la carpeta proyectos/mi-app"
- "Crea una API en Express en proyectos/mi-api y ejecútala"
- "Inicializa un proyecto de Python FastAPI en proyectos/mi-backend"

OpenCode creará los archivos, instalará las dependencias y ejecutará el proyecto — todo dentro de este mismo entorno.
